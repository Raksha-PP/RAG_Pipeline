# RAG_Pipeline

This project implements a simple Retrieval-Augmented Generation (RAG) system that answers questions using content from a specific website. It retrieves relevant information from the webpage and generates answers using a Large Language Model.

## Tech Stack

Python, LangChain, Ollama (Llama3), FAISS, RecursiveCharacterTextSplitter, Nomic Embed Text.

## Workflow

Website → Text Chunking → Embeddings → FAISS Vector Store → Retrieval → LLM Answer.

## Conclusion

This project demonstrates a basic website-based RAG system that retrieves relevant content and generates accurate answers using a local LLM.
